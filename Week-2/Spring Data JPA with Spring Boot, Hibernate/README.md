# Spring Data JPA with Spring Boot, Hibernate

# Hibernate Query Language (HQL) and Native Query

## Project Overview

This project demonstrates the implementation of **Hibernate Query Language (HQL)** and **Native SQL Queries** using **Spring Boot**, **Spring Data JPA**, **Hibernate**, and **MySQL**.

The project is developed as part of the **Cognizant Digital Nurture Program – Java FSE** hands-on exercises.

---

## Technologies Used

- Java 21
- Spring Boot 3.5.16
- Spring Data JPA
- Hibernate ORM
- Maven
- MySQL 8
- IntelliJ IDEA

---

## Database

Database Name:

```
ormlearn
```

Tables Used:

- Employee
- Department
- Skill
- Employee_Skill

---

## Hands-on Implemented

### Hands-on 2 – Get Permanent Employees using HQL

Implemented an HQL query to retrieve all permanent employees from the Employee table.

Features:
- HQL Query
- Fetch Department
- Fetch Employee Skills
- Display Employee Details

---

### Hands-on 3 – Find Average Salary

Implemented an HQL aggregate query to calculate the average salary of all employees.

Features:
- AVG() function
- HQL Aggregate Query

---

### Hands-on 4 – Department-wise Average Salary

Implemented a parameterized HQL query to calculate the average salary of employees belonging to a specific department.

Features:
- Named Parameter
- @Param Annotation
- Department-wise filtering

---

### Hands-on 5 – Get All Employees using Native Query

Implemented a Native SQL Query to retrieve all employees from the database.

Features:
- Native SQL Query
- Spring Data JPA
- Employee Listing

---

### Hands-on 6 – Criteria Query

The provided Cognizant material contains an introduction to Criteria Query and explains its purpose in building dynamic queries. No implementation steps were provided in the exercise document.
